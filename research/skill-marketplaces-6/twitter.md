## Deep-Scraped Articles (Linked from Tweets)

### A. ClawHavoc: 341-1,184 Malicious Skills on ClawHub (The Hacker News, Feb 2, 2026)
**Source:** [thehackernews.com](https://thehackernews.com/2026/02/researchers-find-341-malicious-clawhub.html)

Key findings from Koi Security's audit of 2,857 ClawHub skills:
- **335 skills** used fake prerequisites to install **Atomic Stealer (AMOS)** on macOS
- Attack dubbed **"ClawHavoc"** — targeted macOS users running OpenClaw 24/7 on Mac Minis
- Malicious skills masqueraded as: ClawHub typosquats, cryptocurrency tools, Polymarket bots, YouTube utilities, auto-updaters, Google Workspace tools
- Some skills hid **reverse shell backdoors** inside functional code
- Others exfiltrated bot credentials from `~/.clawdbot/.env` to webhook.site
- **Root cause:** ClawHub is open by default; only restriction is a 1-week-old GitHub account
- **Response:** OpenClaw added a reporting feature; skills with 3+ reports are auto-hidden
- **Palo Alto Networks warned** of "lethal trifecta": private data access + untrusted content exposure + external communication ability
- **Time-shifted prompt injection** via persistent memory: "malicious payloads can be fragmented... written into long-term agent memory, and later assembled into executable instructions"

### B. ClawHavoc Deep Dive: 1,184 Compromised Skills (ClawPort, Feb 17, 2026)
**Source:** [clawport.io](https://clawport.io/blog/clawhavoc-supply-chain-attack-what-happened)

Expanded analysis reveals the attack was even larger than initially reported:
- **1,184 malicious skills** (roughly 1 in 5 packages in the entire ecosystem)
- **Novel attack vector:** Skills didn't just steal data — they **rewrote agent behavior** via persistent memory files
- Three-step pattern: Install normally → Modify memory files → Self-remove traces
- Compromised agents could: forward conversations, alter financial advice, ignore user instructions, exfiltrate API keys
- **Changes survived skill uninstallation, agent restarts, and OpenClaw updates**
- Key distinction: "A compromised npm package can steal environment variables during install. A compromised OpenClaw skill can permanently alter an agent that has access to your email, calendar, CRM, and file systems."
- Memory files are **designed** to be self-modifying — no clear distinction between legitimate and malicious updates

## Analysis: What's New Since Round 5

### 1. Security Crisis Has Deepened Dramatically

**Round 5 status:** VirusTotal partnership announced; security "becoming table stakes."
**Round 6 status:** Full-blown supply chain crisis.

The ClawHavoc incident is the defining event of this period. Key escalations:
- The attack surface expanded from **341 → 1,184 malicious skills** (1 in 5 packages)
- Attack sophistication leaped: from simple credential theft to **persistent memory poisoning** that survives uninstallation
- **Cisco's AI security team** publicly documented data exfiltration via ClawHub skills
- ZoomEye open-sourced a Security Practice Guide in response
- Multiple security firms (Koi Security, Palo Alto Networks, Cisco) now actively researching this space
- New concept: **"time-shifted prompt injection"** — malicious payloads stored in memory, detonating later when conditions align

**Sentiment shift:** In Round 5, security was a "should do." In Round 6, it's an existential threat to marketplace trust. @ZackKorman's contrarian take ("securing a skills marketplace for bots isn't even a top five problem") was met with pushback given the ClawHavoc scale.

### 2. Monetization Models Are Crystallizing

**Round 5 status:** Pay-per-use acknowledged as viable but IP protection unsolved.
**Round 6 status:** Multiple competing models in production.

| Model | Champions | Status |
|-------|-----------|--------|
| **Pay-per-command** | Teneo Protocol (USDC via x402) | Live |
| **Pay-per-use** | Kodeus (on-chain), Dante Reminick (advocacy) | Live |
| **Outcome-based pricing** | Jay Thakrar (pioneered) | Live, spreading |
| **Agent-to-agent payments** | Mastercard AgentPay + Stripe/OpenAI Agentic Commerce Protocol | Live (US cardholders) |
| **Micro-rentals** | Rentaclaw (Solana, $0.00025/tx) | Building |
| **Revenue-sharing** | Virtuals.io ($1M+/month deployed) | Live |
| **Bounty/challenge** | SpoonOS ($5K pool + $50K grants) | Active |
| **Subscription (legacy)** | Being displaced | Declining |

The biggest news: **Mastercard AgentPay going live for all US cardholders** and the Stripe/OpenAI **Agentic Commerce Protocol** with Etsy live and 1M+ Shopify merchants incoming. This is mainstream payment infrastructure meeting agent economies.

### 4. SKILL.md Ecosystem Maturing

The SKILL.md format has become a de facto standard with growing sophistication:
- Clear **MCP vs SKILL.md complementarity** emerging: "MCP servers for integrations, SKILL.md for domain expertise"
- **Scripts + instructions separation** pattern: "Scripts handle computation. Instructions handle judgement."
- Multiple **open-sourced Claude Code configs** shipping pre-configured skill orchestration (PM2 + multi-agent)
- **YAML frontmatter** standardized for skill metadata
- **Desktop interface skills** now possible via same SKILL.md format
- Anthropic added skills to **free plan** — massive democratization signal

### 5. "Agent Maestro" as New Professional Role

@coreyganim's tweet about "Agent Maestros" charging $2K-5K per engagement represents a new labor market forming around skill curation, agent training, and orchestration — not building agents from scratch but composing and tuning them.

### 6. Web3 vs Traditional Split Widening

Two distinct tracks are emerging:
- **Traditional/Enterprise:** Anthropic skills ecosystem, SkillsMP, Skills.SH — focused on developer productivity, SKILL.md standards, MCP integration
- **Web3/Crypto:** SpoonOS, Rentaclaw, Teneo, Kodeus, Phala, Virtuals.io — focused on on-chain payments, token economies, decentralized agent identity, micro-transactions

These worlds are not converging. They share vocabulary ("skills," "agents," "marketplace") but have fundamentally different trust models, governance, and user bases.
