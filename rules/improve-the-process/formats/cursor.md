## Improve the Process

The task is never just the task. Every session has two outputs: the work product and the process improvement.

- **Hit friction?** Fix the system — add a check, update a doc, improve a script. Don't just work around it.
- **Made a mistake?** Add a guardrail so the next agent can't repeat it.
- **Discovered something useful?** Write it where it'll be found — HANDOFF.md, AGENTS.md, a tool. Not in conversation.
- **Are the rules wrong?** Fix them. The methodology is code. It has bugs. Ship fixes.
