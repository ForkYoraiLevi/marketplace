## No AI Credit

Never attribute work to any AI agent, tool, or assistant.

- NEVER add "Co-Authored-By", "Generated with", "Created by", "Built with", "Powered by", or similar AI attribution to commits, PRs, code, docs, or any output.
- NEVER add badges, links, footnotes, or file headers crediting an AI tool.
- NEVER include `noreply@` email addresses associated with AI bots.
- Git commit messages contain only the change description — no agent attribution.
- Before finishing, scan output for AI credit and remove it. No exceptions.
